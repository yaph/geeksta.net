READ THE FUCKING MANUAL - Black Text Design for Computer Geeks

Want a custom version of this design? Send me a request.

RTFM is an initialism for the expression "read the fucking manual" or more in the context of the Unix computer operating system, "read the fucking man page". It is usually expressed when the speaker is irritated by another person's question or lack of knowledge. This design features the text "READ THE FUCKING MANUAL" in a black monospace font with the initial letters bolded to highlight the typically used short form RTFM.
