sudo !! - Run the last command as superuser

Want a custom version of this design? Send me a request.

People who regularly work with the command line inevitably forget to run commands with root privileges or those of another user, when necessary. The command "sudo !!" comes to the rescue in bash. The double exclamation mark repeats the last command and you can simply put sudo in front of it, to run it as the superuser. This design has the bash command "sudo !!" printed in a bold black font and is available on light and white backgrounds on shirts, stickers, mugs and other products.
