Quickly Access the ASCII Table from a Terminal

ASCII is the American Standard Code for Information Interchange, it is a 7-bit code. You can quickly access it from a terminal on Unix-like operating system by typing man ascii in a terminal. It'll show the ASCII character set encoded in octal, decimal, and hexadecimal.
