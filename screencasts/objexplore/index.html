Use Objexplore for Inspecting and Debugging Python Code

Objexplore is a Python package that you can use for inspecting and debugging code. Use it to discover features you weren't aware of, for debugging and understanding code. I learned about this wonderful tool in the Talk Python episode about Rich and Textual, the former being one of the building blocks of Objexplore. In this short screencast I demonstrate inspecting the builtins Python module with Objexplore in an IPython shell.
