A Bookmarklet to Replace Force(s) with Horse(s) based on xkcd 1418

This bookmarklet replaces the words force with horse and forces with horses to make reading sad news a bit more fun. The idea comes from xkcd 1418.

Get the source

The source code for this bookmarklet is on GitHub. I also created a bookmarklet for xkcd 1288, which replaces several words and phrases in the text of a Web page. Have fun!
