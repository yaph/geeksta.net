Asset Management in Python by Robert Kluin and Beau Lyddon

An insightful talk from PyCon 2013 by Robert Kluin and Beau Lyddon covering strategies for managing frontend assets like CSS and JavaScript files in Python projects. The focus is on a generic solution that works with different frameworks like Flask, Django or Pyramid and enables compiling assets as well as using alternate languages like Less, Sass or CoffeeScript.
